 
# splex-big
This is the implementation of FD-TS algorithm for Maximum $s$-plex problem.

## Installation
call make

## Usage
`splex -f <filename> -s <paramete s> -t <max seconds>  [-o optimum object] [-c seed]`

## History
This is the most recent version of the FD-TS code
